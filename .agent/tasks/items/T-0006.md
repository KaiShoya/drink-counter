# [T-0006] details/summary の折りたたみ可能を示す視覚的手がかりが不足

**ID**: T-0006  
**作成日**: 2026-05-06  
**優先度**: Medium  
**種別**: enhancement  
**ステータス**: Backlog  
**担当バージョン**: -  
**関連 Issue**: -

---

## Goal

トップページのカウンター詳細行とアクティビティログが `<details>` で折りたたみ可能なことがユーザーに伝わらない。▶ アイコンや展開状態の視覚フィードバックを追加する。

## Scope（対象ファイル・範囲）

- `app/components/domain/counter/molecules/Row.vue`（カウンター詳細の `<details>`）
- `app/components/domain/activity/molecules/Log.vue`（アクティビティログの `<details>`）

## Out of Scope

- `<details>` の機能変更（折りたたみ自体の動作は変更しない）

## Plan

1. `<summary>` に展開/折りたたみ状態を示すアイコン（`mdi:chevron-down` / `mdi:chevron-up` 等）を追加
2. CSS で `details[open] summary` のアイコンを回転させるトランジションを追加
3. ブラウザデフォルトの `▶` マーカーは `list-style: none` / `::marker { display: none }` で非表示にする

## Verification

- [ ] 閉じている状態で ▶ 相当のアイコンが表示される
- [ ] 開いている状態で ▼ 相当のアイコンが表示される（またはアイコンが回転する）
- [ ] ブラウザデフォルトのマーカー（▶）が重複して表示されない
- [ ] SP / PC 両方で確認

## Notes

- アイコンは既存の `@nuxt/icon` + `mdi` を使用する
- `detail-link` クラスの `text-decoration: underline` は維持してよい
