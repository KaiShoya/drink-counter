## Goal

データ画面で読み込み中・0件・エラー時の表示と復帰導線を統一し、画面停止感をなくす。

## Scope（対象ファイル・範囲）

- `app/pages/(protected)/data/total.vue`
- `app/pages/(protected)/data/monthly.vue`
- `app/pages/(protected)/data/annual.vue`
- `app/components/domain/chart/**`（必要に応じて）

## Out of Scope

- 集計クエリのSQL最適化
- 新しい分析メトリクス追加

## Dependencies

- なし

## Plan

1. 共通状態UI（Loading/Empty/Error）コンポーネント方針を決める。
2. total/annual/monthly に状態分岐を実装する。
3. Error で再試行アクションを接続する。
4. Empty 時に次アクションCTAを表示する。

## Verification

- [ ] 取得中にスケルトンまたは明示的ローディングが表示される
- [ ] 0件時に説明とCTAが表示される
- [ ] 失敗時に再試行で復帰できる
