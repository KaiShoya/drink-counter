## Goal

`SubColumn` の飲み物名 `<span>` のみがクリック対象になっており、スマートフォンでタップが外れやすい。行全体をタップ可能にしてタップ精度を向上させる。

## Scope（対象ファイル・範囲）

- `app/components/domain/counter/atoms/SubColumn.vue`

## Out of Scope

- ± ボタンのタップエリア変更（既存サイズで問題なしと判断）

## Plan

1. `<span class="clickable">` の `@click` を親の `<div>` に移動し、行全体をクリック可能にする
2. `cursor: pointer` と `user-select: none` を `<span>` から親 `<div>` の適切な要素へ移動
3. ± ボタンのクリックと競合しないよう、ボタン列は対象外にする

## Verification

- [x] SP で飲み物名の左右余白部分をタップしても選択（`select()` 呼び出し）が発火する
- [x] ± ボタンを押しても `select()` が呼ばれない（ボタンは別列のため競合なし）

## Work Log

- 2026-05-07: `SubColumn.vue` の `@click="select && select()"` を `<span>` から親 `<div class="column">` に移動
- `.clickable` クラス（`cursor: pointer`, `user-select: none`）も親 div に移動し行全体をタップ可能に

## Notes

- `select` prop が `undefined` の場合は何も起こらない挙動を維持すること
