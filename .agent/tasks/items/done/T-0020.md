## Goal

`labels/[id].vue`（ラベル編集画面）のフォーム下に「このラベルの飲み物」セクションを追加し、ラベル単位でフィルタした飲み物一覧の表示・ドラッグ並び替え・削除・表示切替が行えるようにする。全件一覧を操作する必要がなくなり、並び替えコストを大幅に削減する。

## Scope（対象ファイル・範囲）

- `app/pages/(protected)/labels/[id].vue`（メイン実装）
- `app/stores/pages/labels/edit/state.ts`（`deleteTarget`, `showDeleteModal`, `hasUnsavedSort` 追加）
- `app/stores/pages/labels/edit/actions.ts`（`updateHidden`, `deleteDrink`, `clickDeleteDrinkButton`, `saveSort` 追加）
- `app/stores/pages/labels/edit/getters.ts`（`filteredDrinks` — 現在のラベルIDで絞り込んだ computed）
- `app/stores/pages/labels/edit/index.ts`（新規追加した state/actions/getters を re-export）
- `app/utils/locales.ts`（新規 i18n キー定数追加）
- `i18n/locales/ja.yaml` / `i18n/locales/en.yaml`（翻訳追加）

## Out of Scope

- `drinks/index.vue` の変更（全件一覧は現状維持）
- T-0010（飲み物編集のモーダル化）との統合（独立して実装し、T-0010 完了後に連携を検討）
- 新規飲み物追加フローの刷新（既存の `/drinks/new?labelId=xxx` リンクで代替）

## Dependencies

- **T-0019 完了が前提**（`updateDrinksSort` のスコープ対応が必要）

## Plan

1. **getters に `filteredDrinks` を追加**
   - `useDrinksState` の `drinks` を参照し、`drink_label_id === drinkLabelId.value` でフィルタ
   - 返り値は `computed(() => ...)` でリアクティブに保つ
2. **state に並び替え・削除用の状態を追加**
   - `hasUnsavedSort: ref<boolean>(false)`
   - `deleteTarget: ref<DrinkRow | null>(null)`
   - `showDeleteModal: ref<boolean>(false)`
3. **actions にラベル詳細用の操作を追加**
   - `updateHidden(drink)` — `updateDrinkVisible` を呼び出しトースト表示
   - `clickDeleteDrinkButton(drink)` / `deleteDrink(drinkId, name)` — `deleteDrinkById` を呼び出し
   - `saveSort()` — `updateDrinksSort(filteredDrinks)` を呼び出し（T-0019 対応後のシグネチャ）
   - `onDragEnd()` — `hasUnsavedSort.value = true`
4. **`labels/[id].vue` の template に飲み物セクション追加**
   - フォームと飲み物一覧をセクション区切り線（`<hr>` または Bulma の `divider`）で分離
   - `<draggable>` で `filteredDrinks` を表示（列構成: ハンドル / 名前 / 色 / 量 / 操作）
   - ラベル列は省略（ラベル詳細内なので冗長）
   - フッターに「並び順を保存する」ボタンと「このラベルに追加」リンク（`/drinks/new?labelId=xxx`）
   - 空状態 UI: filteredDrinks が空のとき「まだ飲み物がありません」メッセージ＋追加リンク
5. **未保存状態ガードの統合**
   - `isDirty`（フォーム変更） `|| hasUnsavedSort.value`（並び替え） の両方を `onBeforeRouteLeave` / `beforeunload` で監視
6. **i18n キー追加**
   - `labels.drinks_section_title`（「このラベルの飲み物」）
   - `labels.drinks_empty`（「まだ飲み物がありません」）
   - `labels.drinks_add`（「このラベルに追加」）
   - `labels.save_sort`（「並び順を保存する」）
   - `labels.sort_success`（「並び順を保存しました。」）
