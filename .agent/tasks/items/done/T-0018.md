**優先度**: Low  
**種別**: enhancement  
**ステータス**: Done  
**担当バージョン**: -

## Goal

飲み物一覧・編集フォームにおける細かい UI 問題（タップ領域・ヘッダー欠如・ボタン表示制御・アイコン意味不明確）をまとめて改善し、モバイル使用感と可読性を高める。

## Scope（対象ファイル・範囲）

- `app/pages/(protected)/drinks/index.vue`
- `app/components/domain/drink/atoms/Edit.vue`

## Out of Scope

- 一覧のドラッグ並び替えやソート保存の仕組み（T-0015 で対応）

## Plan

1. **visible トグルのタップ領域拡大**
   - `<span class="icon">` → `<button class="button is-ghost">` に変更
   - `min-width: 44px; min-height: 44px` を保証（WCAG 2.5.5 対応）

2. **操作列にヘッダーを追加**
   - 飲み物一覧・ラベル一覧の列ヘッダー行に「操作」（i18n キー）のテキストを追加

3. **「ラベルのカラーをコピー」ボタンの disabled 廃止**
   - ラベル未選択時は `v-if` で非表示にし、disabled 状態での表示を廃止する

4. **ランダムカラーボタンのアイコン変更**
   - `mdi:cached` → `mdi:dice-multiple` に変更
   - ボタンに `title` 属性（tooltip）でランダム生成の説明を追加

5. **ボタン間の余白追加**
   - 保存ボタンとキャンセルボタンの間に `mr-2` を追加

## Verification

- [x] visible トグルのタップ領域が 44×44px 以上になっている
- [x] 飲み物一覧の操作列に「操作」ヘッダーが表示される
- [x] ラベル未選択時に「ラベルのカラーをコピー」ボタンが表示されない
- [x] ランダムカラーボタンのアイコンがダイスに変わり、tooltip で意味が伝わる
- [x] 保存・キャンセルボタン間に視覚的な余白がある

## Notes

- `<button class="button is-ghost">` は Bulma でボーダーなしのボタンを表現できる
