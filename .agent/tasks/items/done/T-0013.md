## Goal

`/drinks/:id`、`/drinks/new`、`/labels/:id`、`/labels/new` の各ページに `<h1>` 見出しまたはパンくずナビゲーションを追加し、ユーザーが現在地を把握できるようにする。

## Scope（対象ファイル・範囲）

- `app/pages/(protected)/drinks/[id].vue`
- `app/pages/(protected)/drinks/new.vue`
- `app/pages/(protected)/labels/[id].vue`
- `app/pages/(protected)/labels/new.vue`
- `app/utils/locales.ts`（見出しテキストのキー追加）
- `i18n/locales/ja.yaml`
- `i18n/locales/en.yaml`

## Out of Scope

- トップページ・設定ページなどすでに SEO タイトルが設定されているページへの変更

## Plan

1. i18n に `drinks.edit_title`、`drinks.new_title`、`labels.edit_title`、`labels.new_title` キーを追加
2. `locales.ts` に対応する定数を追加
3. 各ページの `<template>` 先頭に Bulma `breadcrumb` または `<h2 class="title is-5">` を追加
4. `useSeoMeta({ title })` も各ページで設定する（現状未設定）

## Verification

- [x] `/drinks/:id` で「◯◯ を編集」のような見出しが表示される
- [x] `/drinks/new` で「飲み物を追加」のような見出しが表示される
- [x] ラベル編集・新規ページも同様に見出しが表示される
- [x] 英語ロケールでも対応するテキストが表示される

## Notes

- 見出しは飲み物名などの動的な値を含めると親切（例: `編集: ビール`）
- パンくずの場合: `飲み物一覧 > 編集` のような2段階で十分
